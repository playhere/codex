# Codex Games

This repository contains small browser-based games.

Open `index.html` in your browser to choose from the available games.

## Snake Game

Open `snake.html` to play a simple Snake game implemented in JavaScript. Collect the red fruit to grow the snake and increase your score.

Use the arrow keys to control the snake. When the game ends, press **Enter** to restart. The snake speeds up slightly every five fruits for an extra challenge.

## Tetris Game

Open `tetris.html` to play a classic Tetris clone. Move pieces with the arrow keys, rotate with **Q** and **W**, and try to clear as many lines as possible. Your score increases by 10 for every line cleared.
